Об программе:
Это сайт Todo — удобное веб-приложение для управления задачами. Здесь вы можете создавать заметки (задачи), редактировать их, удалять и отмечать как выполненные. Выполненные задачи автоматически перемещаются в отдельный список для наглядного контроля.

Помощь:
Краткое руководство:
• Создание задачи: Нажмите на кнопку "Добавить новую задачу", далее заполните все поля и нажмите "Добавить".
• Редактирование: В списке "Не выполненные" будут задачи которые можно отредактировать нажав на кнопку "Изменить".
• Удаление: Для удаления задачи нужно нажать на кнопку "Удалить" и подтвердить действие.
• Отметить как выполненную: В списке "Не выполненные" будут задачи которые можно отметить как выполненные, для этого нужно нажать кнопку "Завершить".
• Просмотр задач: В центре шапки сайта есть две кнопки "Выполненные" и "Не выполненные" с их помощью можно смотреть задачи которые вы выполнили и которые нет.
• Выйти из аккаунта: Для выхода из аккаунта нажмите на свой логин в шапке сайта и подтвердите действие.
